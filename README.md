# FlatBlock

FlatBlock is a compact block puzzle game built around a 9×9 board split into visible 3×3 groups. The goal is to place incoming pieces as efficiently as possible, clear lines and 3×3 squares, and push your score higher without running out of moves.

## How it works

- The board uses a 9×9 layout with visible 3×3 sections.
- Each move places one piece from the current tray onto the board.
- Full rows, full columns, and full 3×3 squares are cleared automatically.
- The game tracks score, combo, and best score during play.
- The run ends when none of the available pieces can be placed anymore.

## Current prototype

The current version is a SwiftUI prototype focused on the core gameplay loop: drag a piece, preview its placement, clear lines or squares, and survive as long as possible while building score and combo.
